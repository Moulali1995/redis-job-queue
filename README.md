# moleculerjs-redis-jwt-session

This project involves basic local authentication (username and password) with JWT and maintaining sessions with redis.

Getting set up :+1:
## Required software/tools
1. Node.js
2. Git
3. VS Code Editor
4. Redis

## Run Project
1. `git clone <git-repo-url>`.
2. `cd <project-folder>`.
3. `npm i`.
4. `npm start`
