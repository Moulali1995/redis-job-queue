# redis-message-queue

Message queues using redis. Create and add jobs to queue and let the redis queue processor do the job.

Getting set up :+1:
## Required software/tools
1. Node.js
2. Git
3. VS Code Editor
4. Redis

## Run Project
1. `git clone <git-repo-url>`.
2. `cd <project-folder>`.
3. `npm i`.
4. `npm start`
